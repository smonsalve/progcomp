La siguiente es la descripción de la Practica 2 del curso de Programación de computadores, esta debe ser entregada máximo el día domingo 15 de Mayo a las 11:59pm a través de la plataforma de EAFIT Interactiva.\\

Suponga que usted es el profesor(a) de una Materia universitara, tal materia tiene \emph{N} cantidad de actividades evaluativas, que se encuentran indexadas en un archivo dado (\emph{indice.txt}).

\section*{Entrada}

Como entradas usted tiene un archivo <<\emph{indice.txt}>> que contiene:
 
\begin{itemize}

\item  la cantidad de archivos
\item  la cantidad de estudiantes
\end{itemize}

 N veces (1 por cada archivo):

\begin{itemize}
\item El nombre de cada archivo
\item Porcentaje actividad evaluativa
\item La cantidad de estudiantes que presetaron esa actividad evaluativa
\item Que estudiantes no presentaron tal evaluación
\item La cantidad de puntos que tenia cada actividad y los porcentajes de cada punto. 
\end{itemize}

\vdots

Adicionalmente, un archivo por cada evaluación donde esta el identificador de cada estudiante y las notas por cada punto. 

Cantidad de Archivos: N\\
Estudiantes totales: M\\[0.5cm]

Parcial1.csv\\
Porcentaje de la evaluacion: [$valor$ $entre$ $1$ y $100$]\\
Presentaron: $v$ $estudiantes$, donde $v \leq M$\\
Total Puntos: $P$\\
Estudiantes que no presentaron: [$valor \leq M$]\\
porcentaje de los puntos: [$P_1, P_2,...,P_N$]

\newpage

\section*{Salida}

Se requiere que usted elabore un programa que cumpla con los siguientes requisitos: 

\begin{enumerate}
\item \textbf{(Valor 25\%)} Manipulación datos:
\end{enumerate}

\begin{itemize}
  \item Leer los archivos a partir del archivo indice, y obtener todos los datos para procesar
  \item Realizar un arreglo con las notas de cada uno de los estudiantes
  \item Realizar un arreglo por estudiante con las notas de cada evaluación
\end{itemize}
 \vspace{0.5cm}

\begin{enumerate}[resume] 
\item \textbf{(Valor 25\%)}Procesamiento
\end{enumerate}

\begin{itemize}
  \item Encuentre el Promedio de las notas de cada estudiante.
  \item Encuentre la desviación típica para cada estudiante.
  \item Encuentre la Nota promedio para cada evaluación
  \item Encuentre la desviación típica por evaluación.
\end{itemize}
\vspace{0.5cm}

\begin{enumerate}[resume]
\item \textbf{(Valor 25\%)} Visualización
\end{enumerate}


\begin{itemize}
  \item Que el programa grafique las notas de cada uno de los estudiantes y los guarde en un archivo con todas las graficas
  \item Graficar el progreso de todos los estudiantes (Gráfica de lineas con linea por cada estudiante) guardarlo como una imagen aparte
\end{itemize}
\vspace{0.5cm}

\begin{enumerate}[resume]
\item \textbf{(Valor 25\%)} Interpretación y Reportes
\end{enumerate}


\begin{itemize}
  \item Determinar cuáles estudiantes han mejorado su rendimiento y mostrarlo mediante una gráfica (solo estudiantes que siempre mejoraron su nota)
  \item Exportar las notas tanto de cada evaluacion como finales en un archivo con formato CSV ordenados de mejor a peor nota final. 
\end{itemize}
